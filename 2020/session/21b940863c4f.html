今だから聞きたい 「一番新しい xRアプリの作り方」 2020年 最新版

講演概要

世界中の AR/VR/MR アプリケーションがUnityで開発されています。そんな中で各社のSDKも日々進化し更にはUnityのxR開発のためのソリューション新しくリリースされています。今や検索すればするほどわからなくなるxRアプリ開発。今回は改めて Unity を使って xRアプリ開発をする方法を少し整理してご紹介します。

講演者詳細

  • 高橋 忍

    Project Manager, Unity Technologies Japan

    新卒で三菱重工業名古屋航空宇宙システム製作所に入社し海上自衛隊のSH-60Kヘリコプタの新規開発の中でも自動操縦システムの開発に従事。その後日本マイクロソフト株式会社ではテクニカルエバンジェリストとして新しい技術を紹介しながら、特にHoloLensの日本での展開の際にはテクニカルリードとして技術情報の紹介からパートナー育成などを担当。現在は Unity Technologies Japanでエンタープライズ向けのプロジェクトマネージャーをしつつ、Unity の新しいサービスをご紹介しています。
