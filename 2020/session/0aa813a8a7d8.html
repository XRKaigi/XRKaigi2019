ぼくたちの「らくがきAR」ができるまで

講演概要

2020年8月1日に、iPhone/iPad用アプリとしてリリースした「らくがきAR」。 リリース直後から、日本だけでなくアジアを中心に7カ国で有料App総合ランキング1位を獲得し、同時に世界各国のメディアにも多数紹介頂き、大きな反響を頂きました。 講演では、その制作の背景と、そこに至るまでのみちのりをご紹介します。

講演者詳細

  • SO

    Designer / Creative Director, Whatever Inc.

    SO Designer / Creative Director 同志社大学卒業後、京都造形芸術大学を卒業。ウェブサイト・インスタレーション制作の企画からアートディレクション、デザインを主にイラストレーション・プログラミング・DTMにも広く浅く携る。2006年10月に株式会社ココノヱを立ち上げ、2019年1月にWhateverに移籍。カンヌライオンズ、AD STARS、ADC等、国内外の賞を受賞。好きな力士は千代の富士。
