講演概要

HoloLensの登場により、多くの企業がMRに有用性を期待し、様々な試行を行っています。さらに今年はHoloLens 2の登場により一層加速しています。
本セッションでは、企業での様々な試行事例を紹介し、企業でのHoloLens活用の動機や課題などをお伝えします。

講演者詳細

  • 中村 薫

    株式会社ホロラボ 代表取締役 CEO

    元は組み込みソフトやWindowsアプリ開発をしていたが、2011年のKinect登場で状況が一変し個人事業主として独立。2016年のHoloLens登場により会社として動いた方がいいという直感がありホロラボを設立する。
    Microsoft MVP for Windows Development。
    著書に「KINECT for Windows SDKプログラミング v2センサー対応版」など 。
